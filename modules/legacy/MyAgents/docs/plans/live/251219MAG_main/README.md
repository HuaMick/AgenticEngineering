# Main Planning: Production State & Post-Deployment Log (MAG)

This directory is a permanent fixture for the `main` branch planning and production state tracking for the `MyAgentsGuidance` (MAG) project.

## Purpose
- **Production Validation**: Track smoke tests and post-deployment verification results.
- **State of Truth**: Maintain the canonical state of the system after staging promotion.
- **Permanent History**: This folder remains constant, providing a historical log of all releases to main.

## Structure
- `live/`: Contains the active state tracking or emergency fix plans for the main branch.
- `logs/pre-deployment/`: Historical record of post-deployment verification runs.
- `results/`: Sign-off reports, release notes, and final deployment audits.
