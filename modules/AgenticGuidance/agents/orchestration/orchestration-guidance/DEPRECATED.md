# DEPRECATED: orchestration-guidance

**Status**: DEPRECATED
**Deprecated**: 2026-01-11
**Replaced By**: orchestration-executor

## Reason for Deprecation

The `orchestration-guidance` agent has been replaced by the generic `orchestration-executor` agent that reads Plan-MMD files dynamically instead of following a hardcoded guidance execution flow.

## Migration Path

1. **Use orchestration-executor** instead of orchestration-guidance
2. **Plan-MMD Required**: Plans must include an `orchestration_*.mmd` file in the plan folder
3. **Generated by orchestration-planning**: The MMD file is now generated by orchestration-planning after all planner phases complete
4. **Dynamic Routing**: Agent routing (including teacher agents) is determined from AGENT_ROUTING metadata in the MMD file

## Reference Documentation

- **Specification**: `modules/AgenticGuidance/assets/specifications/orchestration-executor-specification.yml` (Section 1.2)
- **MMD Schema**: `modules/AgenticGuidance/assets/specifications/plan-mmd-schema.yml`
- **Executor Agent**: `modules/AgenticGuidance/agents/orchestration/orchestration-executor/`

## Historical Context

The `process.mmd` file in this directory shows the original hardcoded guidance orchestration flow. This has been converted to a reference example at:
- `modules/AgenticGuidance/assets/examples/orchestration/orchestration_guidance_reference.mmd`

## Do Not Use

This agent should NOT be invoked directly. All execution flows should use `orchestration-executor` via the `_orchestrate.yml` entrypoint.
